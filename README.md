Shopping Cart Application
Overview

This project is a simple Shopping Cart web application developed using Java Servlets, JSP, JDBC, Oracle SQL, and Apache Tomcat. The application allows users to add products to a shopping cart, view all available items, and remove items from the cart.

The project demonstrates the implementation of CRUD (Create, Read, Delete) operations using JDBC and follows the MVC (Model-View-Controller) architecture.

Features
Display all shopping cart items
Add a new item with name and price
Remove an existing item
Auto-generated Item ID using Oracle Identity Column
Input validation
Responsive user interface using HTML and CSS
Database connectivity using JDBC
Technologies Used
Java
JSP (JavaServer Pages)
Java Servlets
JDBC
Oracle SQL
Apache Tomcat 10
HTML5
